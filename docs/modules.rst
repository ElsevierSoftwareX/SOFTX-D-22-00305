About
=======

The library is directed into performing multi-criteria evaluation based on the fuzzy logic, especially using the Triangular Fuzzy Numbers. Beside the assessment methods, additional techniques are provided like normalizations, defuzzifications, and distance measures. With help of this library, user can calculate the positional ranking of the decision variants in the problems, were data are not exactly known. It can help to find the rational choices between available alternatives and can be used as the tool for creating the decision support systems.
